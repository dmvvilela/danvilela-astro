Swopz Partner it's the storekeeper's app for Swopz.

This is a very simple app, it just validates the vouchers for the consumers.

# Features

- REST APIs
- JWT Login
- Package built_value
- BLoC Pattern
- Provider Package
- RxDart
- Reactivity
- Camera (QR Code reading)
